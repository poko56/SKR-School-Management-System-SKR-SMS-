# SKR-School-Management-System-SKR-SMS-
SKR-SMS คือเว็บแอปพลิเคชันบริหารจัดการสถานศึกษาแบบ SPA ทำงานบนระบบ Firebase แบ่งสิทธิ์ผู้ใช้งาน 3 ระดับ (Super Admin, Admin, Student) สงวนสิทธิ์การสร้างบัญชีผู้ใช้ใหม่ให้เฉพาะ Super Admin เท่านั้น มาพร้อมกระดานประกาศข่าวสารส่วนกลางแบบ Real-time เพื่อการสื่อสารภายในโรงเรียนที่มีประสิทธิภาพและรวดเร็ว
